To make the test of sending mails in the terminal, you can use a Gem called: [`mt`][1]

[1]: https://github.com/soveran/mt

You need to install:

{% highlight bash %}
$ gem install mt
{% endhighlight bash %}

If you are using [dep][1]:

[1]: https://github.com/cyx/dep

{% highlight bash %}
$ dep add mt
$ dep install
{% endhighlight bash %}

Then you can run:

{% highlight bash %}
$ mt 2525
{% endhighlight bash %}
